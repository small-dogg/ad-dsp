# DSP 광고 플랫폼 기능 정의서

## 개요
키워드 및 카테고리 기반 광고 플랫폼. 파트너가 Deal 서비스에 등록된 자사 상품을 광고로 등록하고, 사용자의 키워드 검색 또는 카테고리 탐색 시 광고를 노출한다.

---

## 1. 파트너 관리
파트너(광고주) 계정 및 인증 관리

- 파트너 회원가입/로그인
- 파트너 프로필 관리
- API 키 발급 및 관리
- 파트너 등급/티어 관리
- 파트너 상태 관리 (활성/정지/탈퇴)

---

## 2. Deal 서비스 연동
외부 Deal 서비스로부터 파트너의 상품 정보 조회

- 파트너별 상품 목록 조회 API 연동
- 상품 상세 정보 조회 (이름, 가격, 카테고리, 썸네일 등)
- 상품 상태 동기화 (판매중/품절/종료)

---

## 3. 광고 캠페인 관리
광고 운영의 최상위 단위

- 캠페인 생성/수정/삭제
- 캠페인 기간 설정 (시작일/종료일)
- 일별 예산 한도 설정
- 총 예산 한도 설정
- 캠페인 상태 관리 (진행중/일시정지/종료)
- 캠페인 스케줄링 (요일/시간대별 운영)

---

## 4. 광고 그룹 관리
캠페인 하위의 타겟팅 단위

- 광고 그룹 생성/수정/삭제
- 기본 입찰가 설정
- 타겟팅 조건 설정 (키워드/카테고리)
- 광고 그룹별 예산 분배

---

## 5. 광고 등록
Deal 서비스의 상품을 광고로 등록

- 상품 선택 및 광고 등록
- 광고 상태 관리 (활성/비활성)
- 광고-키워드 연결
- 광고-카테고리 연결

---

## 6. 키워드 관리
키워드 검색 시 광고 노출을 위한 키워드 관리

- 키워드 등록/수정/삭제
- 키워드별 입찰가 설정
- 키워드 매칭 타입 (정확/구문/확장)
- 네거티브 키워드 등록 (광고 제외 키워드)
- 키워드 추천
- 키워드 검색량/경쟁도 조회

---

## 7. 카테고리 타겟팅
카테고리 페이지 진입 시 광고 노출을 위한 설정

- 타겟 카테고리 선택
- 카테고리별 입찰가 설정
- 카테고리 제외 설정

---

## 8. 광고 서빙
실시간 광고 조회 및 노출

- 키워드 기반 광고 조회 API
- 카테고리 기반 광고 조회 API
- 실시간 옥션 처리
- 광고 랭킹 및 정렬
- 품질 점수 계산
- 노출 빈도 제어 (Frequency Capping)
- 필터링 (예산 소진, 비활성, 품절 상품 제외)

---

## 9. 입찰/옥션 시스템
광고 노출 순위 결정을 위한 경매 시스템

- 입찰 방식 (CPC/CPM)
- 실시간 입찰 처리
- 옥션 로직 (2nd Price Auction 등)
- 최소/최대 입찰가 제한
- 자동 입찰 전략

---

## 10. 추적 및 측정
광고 성과 측정을 위한 이벤트 추적

- 노출(Impression) 추적
- 클릭(Click) 추적
- 전환(Conversion) 추적
- 전환 추적 연동 (픽셀/Postback)
- Attribution 모델

---

## 11. 리포팅 및 분석
광고 성과 조회 및 분석

- 실시간 대시보드
- 캠페인/광고그룹/광고별 성과 리포트
- 키워드별 성과 리포트
- 카테고리별 성과 리포트
- 기간별 성과 비교
- 리포트 다운로드 (CSV/Excel)

---

## 12. 정산 및 빌링
광고비 충전 및 정산

- 광고비 충전 (선불)
- 크레딧 잔액 관리
- 일별 광고비 차감
- 정산 내역 조회
- 세금계산서 발행
- 환불 처리

---

## 13. 알림
주요 이벤트 알림 발송

- 예산 소진 임박/완료 알림
- 캠페인 시작/종료 알림
- 잔액 부족 알림
- 알림 채널 (이메일/SMS/웹훅)

---

## 14. 관리자 기능
플랫폼 운영 및 관리

- 파트너 관리 (승인/정지/조회)
- 카테고리 마스터 관리
- 정책 관리 (금칙어, 최소입찰가 등)
- 시스템 설정
- 운영 모니터링 대시보드
- 전체 매출/통계 조회

---

## 15. API
외부 시스템 연동용 API

- 광고 서빙 API (매체/SSP 연동)
- 파트너 관리 API
- 리포팅 API
- 웹훅 (이벤트 콜백)
- Rate Limiting

---

## 16. 공통/인프라
시스템 공통 기능

- 인증/인가 (JWT)
- 권한 관리 (RBAC)
- 감사 로그 (Audit Log)
- 캐싱
- 비동기 처리
- 로깅 및 모니터링
